## 3. ステレオタイプ凡例

- `<<boundary>>`: アクター・外部ファイル / 保存域との境界
- `<<control>>`: コンテキスト解決フローの制御ロジック
- `<<persistent>>`: 永続保存域に由来するデータ（ドリフトスコア情報はサブノードエッジのメタデータとして有向グラフ定義に永続されている）
- 解決済み設定 / 有向グラフ / 上流成果物一覧 / 上流成果物要素 / ガイドライン文書 / カスタム文書 / コンテキスト結果 は実行時に構築・破棄される一時データ
