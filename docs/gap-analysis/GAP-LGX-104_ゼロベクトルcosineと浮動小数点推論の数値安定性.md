Document ID: GAP-LGX-104

# GAP-LGX-104: ゼロベクトルの cosine 計算（0 除算）と浮動小数点推論の値再現性が未定義

**親 TP**: TP-LGX-006
**観点出典**: TP-LGX-006 §2.1 観点 B-04（＋ §2.9 観点 D-01 を統合）
**ステータス**: closed (2026-06-10)
**起票日**: 2026-06-08

## 1. 観点

(1) L2 ノルムが 0 のベクトル（degenerate embedding。空入力等で生じうる）同士、または片方が 0 のペアの cosine 類似度は数学的に未定義（0 除算 → NaN/Inf）になる。REQ.04/REQ.11 はこの数値安定性を規定していない。
(2) 浮動小数点 ONNX 推論は CPU SIMD・スレッド数・BLAS 実装差により同一入力でもベクトル本体がビット非再現になりうる。CTX-INV-1（決定論保証）・SCORE-INV-1 が「値の再現性」まで要求するのか、「順序の決定性」のみかが不明（REQ.04/REQ.11 は順序の決定性のみ保証）。

## 2. 現状の SPEC / UC

SPEC-LGX-006 §3 REQ.04 は「コサイン類似度で計算する」、REQ.11 は「SCORE-INV-1（決定性）を保証する順序で返却」と記述するが、**ゼロベクトルの cosine の値**（0 / NaN / skip）および**推論ベクトル本体の値再現性**は未定義。§4 不変条件表は CTX-INV-1 を REQ.08（トランザクション境界による同一入力→同一結果）経由で「関連」とするが、これは DB トランザクションの話であり、ONNX 推論の浮動小数点決定性を指していない。

## 3. 期待される情報

- ゼロベクトル/ノルム 0 ペアの cosine 定義: 0.0 を返す / skip（次元不一致 skip と同様の集約 Warning）/ NaN を許容しない保証
- 浮動小数点推論の値再現性に対する SPEC レベルの立場:
  - (a) 同一環境内でのビット再現性を保証する（スレッド数固定・確定的 BLAS）
  - (b) 順序の決定性のみ保証し、値の微小差は許容する（その場合 drift_threshold の余裕で吸収する旨）
- CTX-INV-1 が embedding 値に適用されるか否かの明文化（§4 不変条件表の更新）

## 4. 影響範囲

- CTX-INV-1 / SCORE-INV-1 の解釈が下流全体（check の semantic 判定、drift、report、calibrate）の決定性テスト（NFR REL.03/REL.05、SPEC-LGX-010 REQ.06 の出力バイト一致テスト）の合否基準を左右する
- 下流 DD / TS: cosine の数値安定性ガードと、決定性テストの許容誤差（厳密一致 vs εトレランス）
- GAP-LGX-101（空テキスト → ゼロベクトル）と直接連携

## 5. 解決経緯（クローズ時に記入）

SPEC-LGX-006 v0.6.0（人間承認 2026-06-10）: REQ.04 にゼロベクトル cosine の skip + 集約 Warning（standalone drift は Error 維持）と、決定論保証の適用範囲（走査・出力順序のみ、ONNX 推論値のビット再現性は対象外）を確定。§4 CTX-INV-1 行更新。ADR-LGX-003。

## 6. 関連 ADR

浮動小数点決定性の方針（再現性保証 vs εトレランス）は architectural 判断のため **ADR 候補**。GAP-LGX-101 の数値安定性と合わせて 1 ADR で扱いうる。
