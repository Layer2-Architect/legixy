Document ID: GAP-LGX-105

# GAP-LGX-105: cosine 類似度の値域・特異点（完全一致/直交/反対）の定義が未明示

**親 TP**: TP-LGX-006
**観点出典**: TP-LGX-006 §2.1 観点 B-05
**ステータス**: closed (2026-06-10)
**起票日**: 2026-06-08
**重要度**: minor (verification: low-value, 人間判断で drop 可) — 敵対的精査パス 2026-06-09: cosine 値域 [-1,1] は数学的性質であり「決定」ではない。負値の clamp は SPEC-LGX-010 REQ.05 が既に処理済み。明文化の価値はあるが trivial。WEAK_OR_PADDED として維持。

## 1. 観点

REQ.04/REQ.11 は cosine 類似度を計算するとのみ記述し、その値域（[-1.0, 1.0]）と特異点（完全一致 = 1.0、直交 = 0.0、反対 = -1.0、負値の発生可否）の定義が無い。負の cosine の扱いは下流に伝播する（SPEC-LGX-010 REQ.05 の calibrate ヒストグラムは「域外スコア（負の cosine 等）を clamp」と記述しており、SPEC-006 側で負値が発生しうる前提が暗黙にある）。

## 2. 現状の SPEC / UC

SPEC-LGX-006 §3 REQ.04 は「2 ノード間の意味的類似度はコサイン類似度で計算する」のみ。**値域・特異点・負値の発生**を明示していない。SPEC-LGX-010 REQ.05 が calibrate 側で「負の cosine を clamp」と前提しているため、SPEC-006 が負値の発生を正準として認めているかの確認・明文化が必要（責務上 cosine の値域は計算側 = SPEC-006 が定義すべき）。

## 3. 期待される情報

- cosine 類似度の値域 [-1.0, 1.0] の明示
- L2 正規化済みベクトル（REQ.01）前提での内積 = cosine の同値関係
- 負値（意味的反対）の発生可否と、それが正常な出力であること
- 完全一致 1.0 が浮動小数点誤差で 1.0 を超えないことの保証（clamp の要否。GAP-LGX-104 の数値安定性と連携）

## 4. 影響範囲

- SPEC-LGX-010 REQ.05（calibrate の clamp）/ SPEC-LGX-004 REQ.02（閾値判定）の前提となる値域が SPEC-006 で確定しないと、下流の clamp/閾値仕様が宙に浮く
- 下流 DD / TS: cosine 関数の値域不変条件テスト（境界値: 同一ベクトル、直交、反対）

## 5. 解決経緯（クローズ時に記入）

SPEC-LGX-006 v0.7.0（人間裁定 fix・承認 2026-06-10）: REQ.04 に cosine 値域 [-1.0, 1.0]・L2 正規化済みでは内積 = cosine・負値は正常出力・浮動小数点誤差の域外値は clamp を明示。SPEC-LGX-010 REQ.05 の calibrate clamp の前提を計算側で確定。

## 6. 関連 ADR

該当なし（cosine の数学的性質の明文化）。
